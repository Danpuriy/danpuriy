## 🎯 Cómo usarlo

1. **Mirá el video** del tema que te interese
2. **Cloná el repo**: `git clone https://github.com/Danpuriy/danpuriy`
3. **Navegá a la carpeta** del video: `ejercicios/etapa-0/E00-12-primer-binario/`
4. **Hacé los ejercicios** y corre los tests:
   ```bash
   cmake -B build && cmake --build build && ctest
   ```
5. **Compartí tu solución** en Discord cuando la tengas

## 🏆 Niveles

Cada concepto se practica 3 veces, cada vez más profundo:

- **🎯 Teórico** — Respondé preguntas para afirmar lo que viste
- **⚔️ Práctico** — Escribí código que pasa tests automáticos
- **🏆 Proyecto** — Aplicá el concepto en un proyecto real que integra varios temas
