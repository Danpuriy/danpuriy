# Videos — DANPURIY

Cada carpeta acá corresponde a un video publicado. Contiene:

- `metadata.yaml` — título, descripción, tags, estado, links a ejercicios
- `guion.md` — el script del video, como se grabó

Los assets pesados (renders, audio, proyectos de edición) están en `~/danpuriy-prod/`, no en este repo.
